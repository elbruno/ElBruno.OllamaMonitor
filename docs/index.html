ElBruno.OllamaMonitor

Monitor your local Ollama in real time—right from your system tray

A tiny Windows system tray tool that gives you instant visual feedback about your Ollama status, resource usage, and models. Perfect for developers who need to see what's happening under the hood.

💾 .NET 10+ 🪟 Windows 10+ 🚀 Real-time Monitoring
Get Started View on GitHub

See It in Action

Real-time system resource monitoring — CPU, RAM, GPU usage at a glance from your tray

Features

Everything you need to monitor Ollama without cluttering your desktop.

🔍 Status at a Glance

Color-coded system tray icon tells you instantly if Ollama is running, a model is loaded, or there's an error.

⚙️ Model Visibility

See which model is currently loaded and active, directly in the monitor window.

📊 Real-Time Metrics

CPU, RAM, and GPU usage graphs with sparklines show you exactly how much your model is demanding.

🪟 System Tray Integration

Runs quietly in your tray. Click to see details, or use the mini monitor window to keep metrics on top.

⚡ Lightweight & Always-On

Minimal overhead. Starts minimized, configurable refresh rate, and zero fuss.

🛠️ CLI Configuration

Change endpoints, refresh intervals, and thresholds from the command line. Fully scriptable.

Installation

Choose your preferred installation method.

📦 Via NuGet (Recommended)

Install the global dotnet tool:

dotnet tool install --global ElBruno.OllamaMonitor

Then launch anytime:

ollamamon

🔨 From Source

Clone the repository and build:

git clone https://github.com/elbruno/ElBruno.OllamaMonitor.git
cd ElBruno.OllamaMonitor
dotnet build src/ElBruno.OllamaMonitor/
dotnet run --project src/ElBruno.OllamaMonitor/

✅ System Requirements

  • Windows 10 / Windows 11 (requires .NET 10 runtime)
  • Ollama running locally (download here)
  • .NET 10 SDK (to build from source)
  • NVIDIA GPU drivers (optional, for GPU metrics)

Usage Guide

Get the most out of ElBruno.OllamaMonitor.

Starting the App

After installation, launch with:

ollamamon

The app starts minimized to the tray by default. Click the tray icon to open the full details window, or use the tray menu to open the always-on-top mini monitor.

Understanding the Tray Icon

The icon color tells you the status at a glance:

Color Meaning
🟤 Gray Ollama is not reachable
🟢 Green Ollama is running, no model loaded
🔵 Blue A model is currently loaded
🟠 Orange A model is running or high resource usage
🔴 Red Error or Ollama unavailable

Configuration Commands

View current configuration:

ollamamon config

Change Ollama endpoint:

ollamamon config set endpoint http://192.168.1.100:11434

Adjust refresh interval (seconds):

ollamamon config set refresh-interval 5

Reset to defaults:

ollamamon config reset

Reading the Dashboard

  • Ollama Status: Displays the connection status and available API endpoint
  • Current Model: Shows which model is loaded (or "none" if idle)
  • Metrics Sparklines: CPU, RAM, and GPU graphs update in real-time
  • Resource Usage: Peak and current values help you understand the overhead
