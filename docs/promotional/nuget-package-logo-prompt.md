# NuGet Package Logo Prompt

Use this prompt to generate the package logo/icon for **ElBruno.OllamaMonitor**.

## Primary Prompt

```text
Create a clean, modern NuGet package icon for a .NET tool named ElBruno.OllamaMonitor. The icon should feel like a lightweight Windows system tray utility for monitoring a local Ollama runtime.

Visual concept:
- A circular tray-style status icon as the main shape
- A subtle monitor pulse, tiny line chart, or compact telemetry bars inside the icon
- Visual cues for local AI monitoring, but no mascot and no complex illustration
- Strong emphasis on clarity at very small sizes

Style:
- Flat, minimal, geometric, modern
- Professional developer-tool aesthetic
- Crisp edges and simple silhouette
- No gradients unless extremely subtle
- No photorealism
- No 3D rendering

Color direction:
- Primary green for healthy/running state
- Optional blue accent for model-loaded state
- Dark neutral outline or transparent background
- High contrast so the icon is readable on light and dark backgrounds

Composition:
- Centered single icon
- No background scene
- No UI screenshot
- No extra decorative elements
- Must look sharp at 64x64, 128x128, and 256x256

Text rules:
- Do not include words, letters, or captions
- Do not place "NuGet", ".NET", or "OllamaMonitor" inside the image

Output requirements:
- Square format
- Transparent background preferred
- Export-friendly for app/package branding
- Suitable for a NuGet package icon, GitHub social preview crop, and Windows app identity
```

## Negative Prompt

```text
No text, no watermark, no photo background, no laptop scene, no dashboard screenshot, no mascot character, no clutter, no glossy 3D icon, no heavy shadow, no brand logos, no tiny unreadable details.
```

## Recommended Variants

1. **Tray Status Circle** — circular green status icon with a tiny telemetry pulse inside.
2. **Monitor Pulse** — simple monitor/heartbeat glyph merged into a tray icon silhouette.
3. **O + Metrics** — abstract O-shaped mark with a subtle bar chart or line signal.

## Suggested Sizes

- 256x256 master
- 128x128 app/package preview
- 64x64 validation check

## Selection Criteria

Choose the version that:

- still reads clearly at small sizes
- feels like a developer utility, not a consumer app
- suggests monitoring without looking noisy
- works on transparent, white, and dark backgrounds
