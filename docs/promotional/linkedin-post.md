# LinkedIn Post

## Main Text

🟢 Just shipped: **ElBruno.OllamaMonitor** — a tiny Windows tray app to monitor your local Ollama runtime.

No dashboards. No bloat. Just a status icon that tells you:
- ✅ Is Ollama running?
- ✅ Is a model loaded?
- ✅ How much CPU/RAM/GPU is it using?

Glance at your system tray. That's it.

Perfect for:
→ Local AI developers who need quick visibility
→ Demo presenters showing real-time resource impact
→ Anyone running Ollama and curious about overhead

The tray icon changes color:
- 🟤 Gray = offline
- 🟢 Green = running
- 🔵 Blue = model loaded
- 🟠 Orange = high usage
- 🔴 Red = error

Built on .NET 10. Fully configurable. CLI support. Open source.

Install: `dotnet tool install --global ElBruno.OllamaMonitor`

## Variant (More Technical)

🟢 Shipped: **ElBruno.OllamaMonitor** v0.1.0

A .NET 10 system tray app for monitoring local Ollama workloads on Windows.

Features:
• Real-time status icon (5-state FSM)
• Floating details window (CPU, RAM, disk, GPU metrics)
• CLI configuration
• Best-effort NVIDIA GPU tracking
• Copy diagnostics to clipboard

Built for developers. Minimal footprint. Full documentation.
